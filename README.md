# Telemetry Verification Tool
... is a C++ application that parses, validates and analyzes the telemetry generated by an Arduino and a DHT22 sensor. The goal is to build a small telemetry pipeline with packet formatting, log capture, verification, fault detection and reporting, not just to read temperature and humidity values.


Focus: practical c++, embedded data generation, packet design, parsing, validation and debugging

Current status: planning + initial setup
